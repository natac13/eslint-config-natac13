# eslint-config-natac13


Personal eslint-config 
Feel free to use.

## Usage

Install the conventions by running:

```
npm install --save-dev eslint eslint-config-natac13
```

Then add the extends to your `.eslintrc.js`:

```javascript
{
  "extends": "natac13",
  "rules": {
    // your overrides
  }
}
```
